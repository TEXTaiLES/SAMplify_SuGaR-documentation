# SuGaR & PGSR

## SuGaR

SuGaR (Surface-Aligned Gaussian Splatting for Efficient 3D Mesh Reconstruction and High-Quality Mesh Rendering) is used for general-purpose 3D surface reconstruction. It extends 3D Gaussian Splatting by aligning Gaussians with scene surfaces, enabling accurate mesh extraction from multi-view images.

- [Official SuGaR repository (GitHub)](https://github.com/Anttwo/SuGaR)
- [SuGaR project page](https://anttwo.github.io/sugar/)

## PGSR

PGSR (Planar-based Gaussian Splatting Representation) is optimized for flat, thin, or planar structures such as textiles and garments. It is the recommended choice when reconstructing fabric, clothing, or any surface where planarity is important.

- [Official PGSR repository (GitHub)](https://github.com/zju3dv/PGSR)
- [PGSR project page](https://zju3dv.github.io/pgsr/)
